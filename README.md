# op1field

> OP-1 Field algorithmic command center (FIELDBOT v3)

## Controls

- **E1** — select page (DRUM / SYNTH / LFO / PERFORM)
- **E2** — param A (energy/register/shape depth/lane)
- **E3** — param B (swing/contour/depth/centre)
- **K2** — generate / action (page dependent)
- **K3** — advance section / mutate (page dependent)
- **K1 (hold)** — alternate layer for play/stop and reset

## Grid

Pages for drum pattern visualization, synth phrases, LFO waveforms, and live CC control.

## Requirements

- norns
- grid (16×8)
- MIDI device (OP-1 Field on ch1)
- engine: None (MIDI only)
